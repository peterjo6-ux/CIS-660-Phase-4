# CIS-660-Phase-4

1. Download the complaint data at this link: https://files.consumerfinance.gov/ccdb/complaints.csv.zip
2. Download the ipynb file located in the github
3. Unzip the complaint data, and upload the csv to your google drive. Do not place it inside any folders!
4. Load the ipynb file in google colab
5. Run the Code cell in google colab, it may prompt you to give it access to your drive
6. The outputs should be placed in a folder in your google drive
